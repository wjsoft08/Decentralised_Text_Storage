# Decentralised Text Storage

## Project Description
Saving Texts online using IPFS

## Quick start
1. clone repo
2. navigate to /app folder
3. run `yarn install` or `npm install`
4. run `yarn start` or `npm start`

Please use Chrome to view this project.
